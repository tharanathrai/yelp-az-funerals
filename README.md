# yelp-az-funerals
A 2-part analysis of Cemeteries and Funeral Services in the state of AZ, using the Yelp Open Dataset

# Dependencies
Python, Apache Spark, Jupyter Notebook, matplotlib, numpy, seaborn, scipy.stats, nltk

# Analysis
I analyzed the dataset from two lenses:
<ol>
  <li> analyzing businesses to understand the competitive landscape </li>
  <li> analyzing user impact on these businesses </li>
</ol>
